# Контрольная работа 3

В ходе выполнения контрольной работы 3 были выполнены следующие практические задачи:

1. Практическая работа 13:
    - реализовано базовое todo-приложение с хранением задач в localStorage
    - добавлен ServiceWorker для кэширования файлов приложения и его оффлайн работы
2. Практическая работа 14:
    - добавлен WebAppManifest с информацией о приложении
    - добавлены различные варианты иконок приложения
3. Практическая работа 15:
    - сгенерирован сертификат для localhost
    - приложение запущено через веб-сервер с ssl-сертификатом для установления https соединения
    - приложение появляется мгновенно при запуске из кэша
4. Практическая работа 16:
    - добавлена синхронизация задач между клиентами с помощью вебсокета
    - реализована подписка/отписка на уведомления через ServiceWorker
    - реализована рассылка push-уведомлений при добавлении новой задачи
5. Практическая работа 17:
    - добавлена возможность установить напоминание о задаче. По истечении срока в 1 минуту отправляется push-уведомление с напоминанием
    - добавлена возможность отложить напоминание. По истечении срока в 1 минуту снова отправляется push-уведомление с напоминанием
