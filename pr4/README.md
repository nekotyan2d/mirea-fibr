# pr4

Небольшое приложение для управления товарами с фронтом на React и бэком на express

## Как запустить

1. Устанавливаем зависимости бэка
    ```
    cd api
    ```
    ```
    npm i
    ```
2. Запускаем бэк
    ```
    npm run dev
    ```
    Бэк запустится по адресу http://localhost:3000
3. Устанавливаем зависимости фронта
    ```
    cd ..
    ```
    ```
    npm i
    ```
4. Запускаем фронт
    ```
    npm run dev
    ```
    Vite-сервер будет доступен по адресу http://localhost:5173
